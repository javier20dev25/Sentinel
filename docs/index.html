Auditoría de Repositorios y Pull Requests

Sentinel se especializa en asegurar tus repositorios conectándose directamente a tu flujo de trabajo. Al conectar un repositorio, Sentinel examina las propuestas (Pull Requests) enviadas por otros desarrolladores.

Este escaneo ocurre ANTES de que el dueño las visualice o descargue, previniendo que código malicioso se infiltre en componentes críticos.

sentinel-pr-monitor

Monitroreando: @main-core

Nueva PR detectada (#112)...

AMENAZA: Hook malicioso detectado.

Acción: Bloqueo preventivo de merge.

Análisis Inicial y Escaneos Configurables

Al enlazar un repositorio, Sentinel realiza un análisis profundo para verificar vulnerabilidades preexistentes o amenazas inyectadas, reportando de inmediato.

Los intervalos de escaneo son configurables y pueden ejecutarse de forma on-demand en cualquier momento.

audit-config.json

{

"initial_scan": true,

"scan_interval_hours": 24,

"deep_audit": true

}

Bloqueo de Instalaciones Peligrosas

Sentinel detiene de forma proactiva instalaciones de paquetes (como npm) si detecta scripts maliciosos de post-instalación.

Funciona como una barrera impenetrable que mata el proceso instantáneamente para proteger tu entorno local.

npm install

[SENTINEL_INTERCEPT]

Firma maliciosa en react-dom-parser

PROCESO MATADO (SIGKILL)

Protección de Subida y Cuarentena

Durante el push hacia remotos, Sentinel inspecciona los paquetes. Si detecta secretos expuestos (AWS keys, .env) o archivos infectados, bloquea el push de inmediato.

Los archivos sospechosos se mueven a una zona de cuarentena segura. El código inseguro nunca abandona tu máquina.

git push origin main

> Enumerating objects...

> Activando Pre-Push Hook...

SECRETO: AWS_KEY en config.js

CAPACIDADES TÉCNICAS COMPLETAS

Sistema Core Descripción Web Local
Auditoría Global (SGA) Registro inmutable de eventos de seguridad con trazabilidad directa a GitHub commits.
Project Shield (SPS) Hardening automático del entorno local (.npmrc) e inspección estática AST profunda.
Asset Guard (SAG) Interceptor de Git Push para bloqueo preventivo de llaves API, secretos y activos prohibidos.
Análisis Inicial Profundo Escaneo exhaustivo del historial de repositorios al vincularlos por primera vez.
Monitoreo de PRs Externas Supervisión en tiempo real de Pull Requests entrantes antes de interacción local.
Bloqueo de Ejecución (NPM) Terminación de procesos si se detecta malware en scripts de post-instalación de paquetes.
Zona de Cuarentena Segura Aislamiento automático de archivos sospechosos detectados durante el pre-push.
Intervalos Configurables Personalización completa de la frecuencia de escaneo y auditoría persistente.
Identidad Maestro (JWT) Gestión de sesiones endurecidas con Bcrypt y tokens JWT para acceso al backend.
Control Remoto Backend Capacidad de apagado remoto (Shutdown) del proceso servidor desde el dashboard.
