S Sentinel
Source Code

Uncompromising Security for Local Repositories.

Real-time vulnerability scanning, strict command whitelisting, and secret leakage prevention. Driven by a native Electron app and raw performance.

Quick Start

Run Sentinel directly from source. No installer needed.

Terminal 
$ git clone https://github.com/javier20dev25/Sentinel.git
$ cd Sentinel/src/ui
$ npm install
$ npm run electron:dev
