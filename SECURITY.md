# Security Policy

## Supported Versions

The following versions of Sentinel currently receive security updates:

| Version | Supported          |
| ------- | ------------------ |
| 3.0.x   | ✅ Yes             |
| 2.x     | ❌ No              |
| < 2.0   | ❌ No              |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a security vulnerability within Sentinel, please use the following method to report it:

1.  **GitHub Security Advisories**: Navigate to the "Security" tab of this repository on GitHub and select "Report a vulnerability". This is the most secure and efficient way to communicate with the maintainer privately.
2.  If you cannot use GitHub Advisories, please contact the maintainer through their official social profiles linked in the [README](README.md).

### Our Commitment

- We will acknowledge receipt of your report within 48 hours.
- We will provide a preliminary assessment of the vulnerability within 5 business days.
- Once fixed, we will provide credit for the discovery (unless anonymity is requested).
